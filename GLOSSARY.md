# Nonlinear dynamics

Small changes in input yield disproportionate changes in output, eg $$y = x^2$$

# Map

* Time: Discrete
* Model: Difference equation

# Flow

* Time: Continous
* Model: Differential equation

# Difference equation

$$
x_{n+1} = f(x_n)
$$

# Differential equation

$$
\frac{df(x)}{dx} = f(x)
$$
